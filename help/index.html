This experience requires WebGL 2.0 in order to run properly. Currently this is supported by default in Chrome, Microsoft Edge, Firefox and Opera.
If you are visiting this site from Safari, we recomend you to switch to another browser. WebGL 2.0 can also be enabled in Safari trough some wizardy:
1. Check this link and follow the instructions
2. Click Develop in the menu bar
3. Experimental Features > WebGL 2.0
4. Click on WebGL 2.0 to enable it
